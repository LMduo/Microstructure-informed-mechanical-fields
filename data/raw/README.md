# Raw Data

This folder contains the Bruker TI 980 `.hys` nanoindentation files.

- `ti980_hys/`: raw `.hys` files
- `ti980_hys_manifest.csv`: file names, sizes, and SHA-256 checksums

The `.hys` files should be reanalyzed with Bruker TI 980 software before
exporting modulus, hardness, or coordinate-grid CSV files.
